# Titanic Data Analysis (Week 1 Task)

This repository contains the completed deliverables for the Week 1 Titanic Data Analysis project. All required files have been packaged into the attached ZIP archive for review.

## 📦 What's Inside the ZIP File?

Once extracted, the ZIP file contains the following three components:

1. **`titanic_analysis.ipynb` (Jupyter Notebook)**
   * The full Python source code detailing the data cleaning steps (handling missing values for Age, Cabin, and Embarked) and 5 distinct data visualizations built using Matplotlib and Seaborn.
2. **`titanic.csv` (Dataset)**
   * The raw Titanic passenger dataset used as the basis for this analysis.
3. **`Week_1_Task_Report.pdf` (Findings Report)**
   * A structured PDF document summarizing the data cleaning methodology, key visual insights (demographics, survival rates, and correlations), and the final project conclusions.

## 🚀 How to Use

1. Download the ZIP file from this repository.
2. Extract the contents to your local machine.
3. Open the `.ipynb` file in Jupyter Notebook or Google Colab to view or execute the code.
